#### config.ini配置文件说明

- word：提示词
- deadline：截止日期
- font_size：提示词字体大小
- font_color：提示词颜色
- font_family：字体族
- font_bold：是否加粗 [true] / [false]
- shining：时间间隔符是否闪烁 [true] / [false]
- music_volume：倒计时提示铃声音量 [0-100]
- top_hint：倒计时置顶显示 [true] / [false]
- word_start：大纲单词起始
- word_end：大纲单词结束
- 修改配置文件必须严格按照原有格式，如果格式错误可以删除配置文件并重启程序
